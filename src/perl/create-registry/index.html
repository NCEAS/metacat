Welcome to the <@organization@> Data Registry. This is the primary source for comprehensive information about scientific and research data sets collected within or under the auspices of the <@orgabbrev@>.

 

This project is a cooperative effort of <@orgabbrev@>, the National Center for Ecological Analysis and Synthesis (NCEAS) and the Center for Environmental Analysis (CEA-CREST). The Data Registry is based on software developed by the Knowledge Network for Biocomplexity (KNB), and houses metadata that are compliant with Ecological Metadata Language (EML).

 

Credit for the datasets in this registry goes to the investigators who collected and processed the data.Our particular thanks go out to the <@orgabbrev@> scientists, and NCEAS staff for their comments and continuing support.

 

Registry Tools

  • Browse existing <@orgabbrev@> data sets
    The registry search system is used to locate data sets of interest by searching through existing registered data sets. Presently the search covers all fields, including author, title, abstract, keywords, and other documentation for each dataset. (More sophisticated search capabilities, including boolean field searches, will be available in future.)

  • Register a new <@orgabbrev@> data set
    The registration page is used to submit information about a new data set associated with <@organization@> research. The documentation about the data set will be reviewed and then submitted to the Registry.
